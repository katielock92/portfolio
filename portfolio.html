API Webserver: ATS

For my major coding assignment for the second Coder Academy term, my task was to create an API webserver for a real-world app idea using Python, Flask and an ORM. I had three weeks to select an app idea, submit a draft ERD for approval, develop and document the database and the APIs. I opted to create a webserver for a basic Applicant Tracking System (ATS).

Terminal Application: Quiz

For our third assignment of the first Coder Academy term, we had two weeks to design, develop and test a terminal application in Python. I opted to create a Rules of Ultimate quiz - designed to emulate the real WFDF Rules Accreditation website and quiz.

Portfolio Website

For my first assignment as part of my Coder Academy bootcamp, I was tasked with creating a personal portfolio website from scratch using the skills that we had learnt in our first month of the course.

Diversity in Tech Scholarship
