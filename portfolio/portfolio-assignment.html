Grade: 92% (High Distinction)

About

For my first assignment as part of my Coder Academy bootcamp, I was tasked with creating a personal portfolio website from scratch using the skills that we had learnt in our first month of the course. I had a slight advantage in this assignment, due to the fact that scholarship submission was a very similar website. However I knew to do the assignment justice, I did have to build something new from scratch - though I was able to leverage some coding components, and adjust accordingly.

The brief for this assignment specified a minimum number of HTML pages, textual components and graphical components that needed to demonstrate our skills, experience and personality. The site also needed to have a responsive design built using CSS flexbox or grid, and include a navigation menu/bar.

In addition to building and deploying this website, a thorough README.md was required to be created, along with a slide deck and presentation summarising our experience building this site.
