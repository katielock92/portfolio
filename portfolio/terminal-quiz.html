Terminal Application: Quiz

April-May 2023

About

After beginning to learn Python only a month prior, for our third assignment of the first Coder Academy term, we had two weeks to design, develop and test a terminal application in Python.

Our application was required to have at least three features, involve file handling and user input, utilise multiple Python libraries and have multiple successful unit test cases. For my application, I opted to create a Rules of Ultimate quiz - designed to emulate the real WFDF Rules Accreditation website and quiz.

My application's features include a user registration and login process, viewing previous results - along with the main quiz itself. Every quiz attempt presents a random sample of true/false questions that are then validated and scored.

The application runs locally through your machine's terminal application, and can be downloaded from my GitHub.
